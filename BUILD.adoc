== Generate Custom CSS

In order to generate custom CSS we have to use http://asciidoctor.org/docs/user-manual/#stylesheet-factory[`stylesheet-factory`]

This script clones the factory repository, installs the dependencies and generates CSS based in the `unifly.scss`

[source,bash]
----
./build_css.sh
----
